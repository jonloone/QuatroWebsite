## Framework Architecture Applied

All documents follow the prescribed messaging frameworks:

**Hub Page (1 document):**
- JTBD + 5-Second Clarity + PAS Hybrid framework
- 11 sections with framework annotations
- Hero passes 5-second test
- Solution menu visible above fold
- Problem narrative with escalating consequence
- Compound value (knowledge graph)
- Trust metrics and audit score table

**Sub-Pages (4 documents):**
- 8-Section Sub-Page Messaging Hierarchy framework
- Each section explicitly mapped to framework principle
- Hero names both solution AND vertical
- Problem depth quantified with specific data points
- How It Works shows team perspective (not architecture)
- Proof includes named products/pricing and cross-industry validation
- Objections woven as confident statements (not FAQ)
- Adjacent solutions shown with compound value
- CTA is 4–5 words, verb-first, specifics below button
- Cross-vertical credibility to adjacent industries
